Chess Game
O projeto ChessGame tem como base uma conexão entre diferentes usuários, que ao ser estabelecida é possível jogar uma partida de xadrez, sua lógica foi baseada em JavaScript utilizando conceitos de POO (Programação Orientada a Objetos).
Conforme o supracitado, a comunicação e renderização da página foi feita com Django como base, que é basicamente é um framework Web Full Stack, open source (código aberto) baseado em Python, ele foi responsável por integrar o WebSocket utilizando o projeto channels responsável por ampliar a comunicação base do django utilizando ASGI,com essa comunicação é possível ter essa conversa entre diferentes usuários.
Além disso, vale ressaltar que o vídeo acima foi utilizando a mesma máquina, porém, em navegadores diferentes para exemplificar a conexão.
