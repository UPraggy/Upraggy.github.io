Gera FolhaAnalitica
App desenvolvido com objetivo de auxiliar na conversão de uma relação de folhas de pagamento de TXT para Excel, filtrando campos necessários, tendo meta inicial de cumprir o Projeto de Pesquisa e desenvolvimento, da matéria Programação Orientada A Objetos Em Java.
Conforme o supracitado, para o programa foi utilizada a linguagem Java e a biblioteca POI responsável por gerar arquivos em Excel.
Em resumo, o projeto foi feito em conjunto com um cliente e um grupo de trabalho, possibilitando experienciar desde o processo de análise de requisitos até a entrega e avaliação de funcionalidade do projeto final.
