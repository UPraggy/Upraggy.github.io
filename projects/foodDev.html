FOOD DEV
O projeto FodDev tem como base o gerenciamento de pedidos, cadastro de clientes e estoque, disponibilizando os dados em Excel e gráficos de estoque, compras e etc.
Conforme o supracitado, o programa foi baseado em Python, a interface foi feita utilizando PyQt5, os sistemas de arquivos foi utilizando a biblioteca Pandas e a geração de gráficos utilizando a biblioteca Matplotlib. Além disso, varias outras bibliotecas e módulos foram utilizados.Além disso, vale ressaltar que o sistema é totalmente offline, visto que foi um projeto copiloto, afim de adentrar o campo da programação e colocar em prática todo o conhecimento adquirido e aperfeiçoa-lo.
Com base nisso, como sendo um dos primeiros projetos, ainda há diversas melhorias possíveis para implatação, e uma delas seria uma base de dados online e a possibilidade do cliente realizar um pedido.
