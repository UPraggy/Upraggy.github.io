Documentations
Website criado com objetivo de divulgar algumas das minhas documentações, originadas de anotações de diversas linguagens e tecnologias aprendidas ao decorrer do tempo.
Nessa perspectiva, cada documentação foi feita com base em um template automático, gerada pela conversão do MarkDown feito no github para html, além de conter algumas formatações em css por minha parte.
Ainda baseado na conversão, o novo html gerado é inserido de forma dinâmica na pagina, contendo índices para navegação e o conteúdo, feito através do JavaScript, onde se encontra um array com parâmetros passados via url.
Exemplo.:
   document.html?doc=react&img=react

Neste exemplo o doc se refere ao html e img a imagem.
